# 🛡 SafeHer – AI Powered Women Safety System

## 📌 Overview
SafeHer is an AI-powered women safety platform designed to provide quick emergency assistance and improve personal safety.  
The system includes an SOS alert feature that sends emergency notifications and the user's location to saved contacts.

It also includes a risk analysis model that predicts safety risk levels (Low, Medium, High) based on user input data.

---

## 🚀 Features

- SOS emergency alert system
- Sends location to saved emergency contacts
- AI-based risk level prediction (Low / Medium / High)
- Secure login authentication
- Emergency contact storage
- User dashboard for quick access to safety tools

## ⚙️ How to Run the Project

Clone the repository

```bash
git clone https://github.com/your-username/safeher.git
```

Open the project folder

Install dependencies

```bash
pip install flask
```

Run the application

```bash
python app.py
```

Open in browser

```
http://127.0.0.1:5000
```
